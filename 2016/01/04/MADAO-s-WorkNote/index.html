MADAO's WorkNote

2016-01-04

  • 对于单例的获取，最好不好通成[DataManager sharedInstance]来获取单例。可以在单独写一个get方法:
    -(DataManager *)dataManager
    { 
    return [DataManager shareInstance];
    }
  • 对于利用NSTimer进行轮播的Banner，可以将视图控制器ViewWillAppear和ViewWillDisapper中进行定时器的注册和销毁，防止内存泄露。
  • 使用MJRefresh时需要注意一点，如果TableView初始化时，并没有Cell,则可能无法生成mj_footer

2016-01-05

  • 谨记MVC模式或者学习使用MVVM模式，不要因为控件和模型相对独立，就将M与V写在一起

2016-01-06

  • 更正:MJRefresh之所以没有mj_footer生成,是因为新功能。需要设置
    self.mainTableView.mj_footer.automaticallyHidden = NO;
    即可解决。
